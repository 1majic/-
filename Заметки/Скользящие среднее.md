# Скользящие среднее
Это самый простой трендовый индикатор. Он представляет собой линию из средних цен.

Может быть:
 - Простым
 - Экспоненциональным
 - Суммой двух

Основные задачи скользящей средней:
- Определени направления тренда
- Подтверждение смены тренда
- Формирование линий поддержки
